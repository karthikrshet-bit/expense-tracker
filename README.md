# 💸 Spend - Expense Tracker

A modern expense tracking web application built using HTML, CSS, and JavaScript.

# Features

- Add income and expense transactions
- Weekly and monthly analytics charts
- Interactive pie charts using Chart.js
- Transaction history grouped by date
- LocalStorage data persistence
- Responsive and modern UI design

# Tech Stack

- HTML
- CSS
- JavaScript
- Chart.js
- GitHub Pages
